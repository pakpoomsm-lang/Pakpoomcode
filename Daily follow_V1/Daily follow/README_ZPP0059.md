# ปุ่ม ZPP0059 — ดึงข้อมูลจาก SAP อัตโนมัติ

ปุ่ม **ZPP0059** (สีเขียว) ในหน้า `daily_follow.html` จะสั่งให้ SAP รัน transaction
ZPP0059 แล้วดึงยอดความคืบหน้าการผลิตเข้ามาอัปเดตตารางให้อัตโนมัติ โดยไม่ต้อง
export ไฟล์เองแล้วกด Update เอง

เนื่องจาก browser รันสคริปต์ SAP เองไม่ได้ (ความปลอดภัย) ปุ่มจึงทำงานผ่าน
โปรแกรมเล็ก ๆ บนเครื่องชื่อ `serve_daily_follow.py`

## สิ่งที่ต้องมี
- Windows ที่ติดตั้ง **Python**
- **SAP GUI** เปิดและ login ค้างไว้ + เปิดใช้งาน **SAP GUI Scripting**
- เป็น **เครื่องเดียวกัน** กับที่เปิดหน้า dashboard

## วิธีใช้
1. ดับเบิลคลิก **`Start_Daily_Follow.bat`** (หรือสั่ง `python serve_daily_follow.py`)
2. หน้าเว็บจะเปิดที่ `http://127.0.0.1:8059/daily_follow.html`
3. เปิด SAP GUI ทิ้งไว้ที่หน้าจอเลือกของ ZPP0059
4. กดปุ่ม **ZPP0059** สีเขียว → รอสักครู่ (อย่าแตะหน้าต่าง SAP) → ตารางอัปเดตเอง

## การตั้งค่า (แก้ในไฟล์ `serve_daily_follow.py`)
- `DYNAMIC_DATES` — `True` = ใส่ช่วงวันที่ให้อัตโนมัติ (ย้อนหลัง 3 วันถึงวันนี้),
  `False` = ใช้วันที่ตามที่บันทึกไว้ในสคริปต์เดิม
- `DATE_FROM_OFFSET_DAYS` / `DATE_TO_OFFSET_DAYS` — ปรับช่วงวันที่
- `EXPORT_DIRS` — โฟลเดอร์ที่ SAP เซฟไฟล์ export ออกมา
  ถ้ากดปุ่มแล้วขึ้นว่า "found no exported file" ให้เพิ่ม path โฟลเดอร์ที่ SAP
  เซฟไฟล์ลงในรายการนี้
- `PORT` — เปลี่ยนพอร์ตได้ถ้าชนกับโปรแกรมอื่น

> หมายเหตุ: ไฟล์ที่ SAP export ออกมาจะถูกคัดลอกทับ `ZPP0059.xlsx` ทุกครั้งที่กดปุ่ม
