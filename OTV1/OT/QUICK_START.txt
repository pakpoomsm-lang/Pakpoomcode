========================================
 คู่มือเริ่มต้นใช้งานด่วน
 ระบบบันทึกการทำ OT พนักงาน
========================================

⚡ วิธีเริ่มต้นใช้งาน (3 ขั้นตอน)

1. เริ่มโปรแกรม
   - Double-click ที่ไฟล์ "run_ot_system.bat"
   หรือ
   - เปิด Command Prompt แล้วพิมพ์: python ot_recording_system.py

2. นำเข้าข้อมูลพนักงาน (ครั้งแรกเท่านั้น)
   - คลิกแท็บ "จัดการพนักงาน"
   - คลิกปุ่ม "เลือกไฟล์ Excel"
   - เลือกไฟล์ "รายชื่อพนักงาน 05_2026.xlsx"
   - รอจนกว่าจะแสดงข้อความ "นำเข้าข้อมูลพนักงานสำเร็จ"

3. เริ่มบันทึก OT
   - คลิกแท็บ "บันทึกการทำ OT"
   - เลือกวันที่, GL, Process, และ Shift
   - Double-click ชื่อพนักงาน → กรอกรายละเอียดงาน → บันทึก
   - คลิก "Preview" เพื่อตรวจสอบ
   - คลิก "บันทึกข้อมูล" เพื่อบันทึกลงฐานข้อมูล

========================================
 การบันทึก OT แบบละเอียด
========================================

ขั้นตอนการบันทึก OT:

1. เลือกวันที่ทำงาน
   └─ คลิกที่ช่อง "วันที่ทำงาน" เพื่อเลือกวันที่

2. กรองพนักงาน (ไม่บังคับ)
   ├─ เลือก GL: เลือกหัวหน้างาน หรือ "All" สำหรับทั้งหมด
   ├─ เลือก Process: เลือกสถานที่ทำงาน หรือ "All"
   └─ เลือก Shift: A, B, D, หรือ "All"

3. เพิ่มรายการงาน
   ├─ Double-click ที่ชื่อพนักงาน
   ├─ กรอกรายการงาน (บังคับ)
   ├─ กรอก Remark (ไม่บังคับ)
   ├─ เลือกสถานะ: NT, Leave, MCP, Subcon
   └─ คลิก "บันทึก"

4. เพิ่มพนักงานคนอื่นๆ
   └─ ทำซ้ำขั้นตอนที่ 3 สำหรับพนักงานทุกคนที่ทำ OT

5. ตรวจสอบข้อมูล
   └─ คลิก "Preview" เพื่อดูภาพรวมก่อนบันทึก

6. บันทึกข้อมูล
   └─ คลิก "บันทึกข้อมูล" เพื่อบันทึกลงฐานข้อมูล

========================================
 การดูรายงาน
========================================

1. คลิกแท็บ "รายงาน OT"

2. เลือกช่วงวันที่
   ├─ จากวันที่: วันที่เริ่มต้น
   └─ ถึงวันที่: วันที่สิ้นสุด

3. เลือก Process (ไม่บังคับ)
   └─ เลือก "All" เพื่อแสดงทุก Process

4. คลิก "ค้นหา"

5. Export เป็น Excel (ไม่บังคับ)
   ├─ คลิก "Export Excel"
   └─ เลือกตำแหน่งที่จะบันทึกไฟล์

========================================
 คำอธิบายสถานะ
========================================

NT      = Normal Time (เวลาปกติ)
Leave   = ลา
MCP     = พนักงาน MCP
Subcon  = พนักงาน Subcontract

========================================
 ปุ่มสำคัญ
========================================

หน้าบันทึก OT:
  [ลบรายการ]      ลบรายการที่เลือกออกจากตาราง
  [Preview]       แสดงภาพรวมก่อนบันทึก
  [บันทึกข้อมูล]   บันทึกลงฐานข้อมูล
  [ล้างข้อมูล]     ล้างรายการทั้งหมดในตาราง

หน้ารายงาน:
  [ค้นหา]         ค้นหาข้อมูลตามเงื่อนไข
  [Export Excel]  ส่งออกเป็นไฟล์ Excel

หน้าจัดการพนักงาน:
  [เลือกไฟล์ Excel]            นำเข้าข้อมูลพนักงาน
  [ลบข้อมูลพนักงานทั้งหมด]     ล้างข้อมูลพนักงาน

========================================
 เคล็ดลับ
========================================

💡 Double-click ชื่อพนักงานเพื่อเพิ่มรายการงานอย่างรวดเร็ว

💡 ใช้ Preview ก่อนบันทึกทุกครั้ง เพื่อตรวจสอบความถูกต้อง

💡 Export รายงานเป็น Excel เพื่อนำไปประมวลผลต่อ

💡 สำรองไฟล์ ot_records.db เป็นประจำ

========================================
 การแก้ไขปัญหาเบื้องต้น
========================================

❌ โปรแกรมไม่เริ่มต้น

❌ ข้อมูลไม่แสดงในรายงาน
   → ตรวจสอบช่วงวันที่ที่เลือก
