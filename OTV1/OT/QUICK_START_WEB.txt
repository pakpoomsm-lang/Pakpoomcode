========================================
 คู่มือเริ่มต้นใช้งานด่วน
 ระบบบันทึกการทำ OT (Web Version)
========================================

⚡ วิธีเริ่มต้นใช้งาน (3 ขั้นตอน)

1. เริ่มเว็บเซิร์ฟเวอร์
   - Double-click ที่ไฟล์ "run_web.bat"
   หรือ
   - เปิด Command Prompt แล้วพิมพ์: python app.py

2. เปิดเว็บเบราว์เซอร์
   - เปิด Chrome, Edge, หรือ Firefox
   - ไปที่ http://localhost:5000

3. นำเข้าข้อมูลพนักงาน (ครั้งแรกเท่านั้น)
   - คลิกเมนู "จัดการพนักงาน"
   - คลิกปุ่ม "เลือกไฟล์ Excel"
   - เลือกไฟล์ "รายชื่อพนักงาน 05_2026.xlsx"
   - คลิก "นำเข้าข้อมูล"

========================================
 การบันทึก OT (5 ขั้นตอน)
========================================

1. คลิกเมนู "บันทึก OT"

2. เลือกวันที่และกรองพนักงาน
   ├─ เลือกวันที่ทำงาน
   ├─ เลือก GL (หรือ All)
   ├─ เลือก Process (หรือ All)
   └─ เลือก Shift (A/B/D หรือ All)

3. เพิ่มรายการงาน
   ├─ คลิกปุ่ม + ที่แถวพนักงาน
   ├─ กรอกรายการงาน (บังคับ)
   ├─ กรอก Remark (ไม่บังคับ)
   ├─ เลือกสถานะ (NT/Leave/MCP/Subcon)
   └─ คลิก "บันทึก"

4. ทำซ้ำขั้นตอนที่ 3 สำหรับพนักงานคนอื่นๆ

5. ตรวจสอบและบันทึก
   ├─ คลิก "Preview" เพื่อดูภาพรวม
   └─ คลิก "บันทึกข้อมูล" เพื่อยืนยัน

========================================
 การดูรายงาน (4 ขั้นตอน)
========================================

1. คลิกเมนู "รายงาน"

2. เลือกช่วงวันที่
   ├─ จากวันที่: เลือกวันที่เริ่มต้น
   └─ ถึงวันที่: เลือกวันที่สิ้นสุด

3. เลือก Process (ไม่บังคับ)

4. คลิก "ค้นหา"

5. Export Excel (ไม่บังคับ)
   └─ คลิก "Export Excel" เพื่อดาวน์โหลดไฟล์

========================================
 เมนูหลัก
========================================

📝 บันทึก OT
   → บันทึกการทำงาน OT ของพนักงาน

📊 รายงาน
   → ดูและ Export รายงาน OT

👥 จัดการพนักงาน
   → นำเข้าและจัดการข้อมูลพนักงาน

========================================
 คำอธิบายสถานะ
========================================

🔵 NT      = Normal Time (เวลาปกติ)
🟡 Leave   = ลา
🟢 MCP     = พนักงาน MCP
🔷 Subcon  = พนักงาน Subcontract

========================================
 ปุ่มสำคัญ
========================================

หน้าบันทึก OT:
  [+]             เพิ่มรายการงาน
  [Preview]       แสดงภาพรวม
  [บันทึกข้อมูล]   บันทึกลงฐานข้อมูล
  [ล้างข้อมูล]     ล้างรายการทั้งหมด
  [🗑️]            ลบรายการ

หน้ารายงาน:
  [ค้นหา]         ค้นหาข้อมูล
  [Export Excel]  ส่งออกเป็น Excel

หน้าจัดการพนักงาน:
  [นำเข้าข้อมูล]              นำเข้าจาก Excel
  [ลบข้อมูลพนักงานทั้งหมด]     ล้างข้อมูลพนักงาน
  🔍                          ค้นหาพนักงาน

========================================
 เคล็ดลับ
========================================

💡 ใช้งานบนหน้าจอใหญ่เพื่อประสบการณ์ที่ดีที่สุด

💡 ตรวจสอบข้อมูลด้วย Preview ก่อนบันทึกทุกครั้ง

💡 Export รายงานเป็น Excel เพื่อประมวลผลต่อ

💡 ค้นหาพนักงานได้โดยพิมพ์รหัส, ชื่อ, หรือสถานที่ทำงาน

💡 ใช้งานได้บนมือถือและแท็บเล็ต (Responsive Design)

💡 เก็บไฟล์ ot_records.db ไว้เป็นสำรอง

========================================
 URL สำคัญ
========================================

เว็บหลัก:        http://localhost:5000
บันทึก OT:       http://localhost:5000/
รายงาน:          http://localhost:5000/report
จัดการพนักงาน:   http://localhost:5000/employees

========================================
 การแก้ไขปัญหาเบื้องต้น
========================================

❌ เซิร์ฟเวอร์ไม่เริ่มต้น
   → รันคำสั่ง: pip install flask pandas openpyxl

❌ เข้าเว็บไม่ได้
   → ตรวจสอบว่าเซิร์ฟเวอร์กำลังรันอยู่
   → ลองเข้า http://127.0.0.1:5000

❌ ไม่สามารถนำเข้าข้อมูลพนักงาน
   → ตรวจสอบไฟล์ Excel ว่ามีโครงสร้างถูกต้อง

❌ ข้อมูลไม่แสดงในรายงาน
   → ตรวจสอบช่วงวันที่ที่เลือก
   → ตรวจสอบว่ามีข้อมูลในช่วงนั้นจริง

❌ Port 5000 ถูกใช้แล้ว
   → แก้ไขไฟล์ app.py เปลี่ยน port เป็น 5001

========================================
 ข้อควรระวัง
========================================

⚠️ อย่าปิดหน้าต่าง Command Prompt ขณะใช้งาน

⚠️ อย่าบันทึกข้อมูลซ้ำสำหรับวันเดียวกัน

⚠️ สำรองฐานข้อมูล ot_records.db เป็นประจำ

⚠️ ตรวจสอบข้อมูลก่อนบันทึกทุกครั้ง

========================================
 ข้อดีของ Web Version
========================================

✅ ใช้งานผ่านเว็บเบราว์เซอร์
✅ UI สวยงามและทันสมัย
✅ Responsive (รองรับทุกหน้าจอ)
✅ ค้นหาข้อมูลแบบ Real-time
✅ Export Excel ได้ทันที
✅ ไม่ต้องติดตั้งโปรแกรมพิเศษ

========================================
 การหยุดเซิร์ฟเวอร์
========================================

กด Ctrl+C ในหน้าต่าง Command Prompt

========================================
 การเข้าถึงจากเครื่องอื่น (LAN)
========================================

1. เช็ค IP ของเครื่อง Server
   > ipconfig (Windows)

2. เครื่องอื่นเข้าผ่าน
   http://<IP-Address>:5000

   ตัวอย่าง: http://192.168.1.100:5000

หมายเหตุ: ตรวจสอบ Firewall ว่าอนุญาต port 5000
