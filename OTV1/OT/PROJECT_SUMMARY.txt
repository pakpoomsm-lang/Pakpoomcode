========================================
 🗄️ Database Schema
========================================

Table: employees
  - emp_id (PRIMARY KEY)
  - first_name
  - last_name
  - workplace
  - emp_type (MCP/SUB)
  - shift (A/B/D)
  - supervisor (GL)
  - gl_number

Table: ot_records
  - id (PRIMARY KEY)
  - work_date
  - emp_id (FOREIGN KEY)
  - workplace
  - work_description
  - remark
  - status (NT/Leave/MCP/Subcon)
  - created_at

========================================
 📊 API Endpoints
========================================

GET  /                           หน้าบันทึก OT
GET  /report                     หน้ารายงาน
GET  /employees                  หน้าจัดการพนักงาน

GET  /api/supervisors            รายชื่อ GL
GET  /api/workplaces             รายชื่อสถานที่ทำงาน
GET  /api/employees/filter       กรองพนักงาน
GET  /api/employees/list         รายชื่อพนักงานทั้งหมด
GET  /api/employees/count        จำนวนพนักงาน
POST /api/employees/import       นำเข้าข้อมูลพนักงาน
POST /api/employees/clear        ลบข้อมูลพนักงาน

POST /api/ot/save               บันทึก OT
GET  /api/reports/search        ค้นหารายงาน
GET  /api/reports/export        Export Excel
